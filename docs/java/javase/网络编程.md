### URL和URI的区别？

二者都可以唯一确定一个网络上的资源，URl是URi的子集。

uri：统一资源标志符，可以是编号，例如人的身份证。

url：统一资源定位符，只能是地址，例如人的家庭地址。

找具体的uri可以通过url
