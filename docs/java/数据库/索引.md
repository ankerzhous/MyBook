如果没有索引，查找一个数据需要把整张表遍历一遍，而有索引，可以减少需要遍历的数量

当使用主键和唯一键创建表时，mysql自动创建名为primary的特殊索引，该索引成为聚簇索引，其他索引称为二级索引或者非聚簇索引。
