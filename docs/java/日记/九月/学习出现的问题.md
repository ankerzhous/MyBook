#### 2.焦虑？

来源：压力，自我对未知困难的无限放大，缺少理性的思考。

案例1：

​	问题来源：要面试了，复习一下数据库，却感觉什么也不会，要不要从头开始，打开B站狂神的视频，握草这么多，哪里看得完，遂放弃，太难了，太多了，还要反反复复学，当时情绪也有问题？

​	怎么解决：

+ 既然是复习，哪怕再怎么样也要在原有的知识体系上改变，而不是从新开始，找到以前的笔记（笔记太重要了）对着标题回忆和讲述。
+ 知识点串了一遍，感觉很棒。
